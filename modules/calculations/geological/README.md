# Расчётный модуль: Геологические изыскания

## Описание

Модуль выполняет расчёт стоимости инженерно-геологических изысканий **строго по СЦИ РК 8.03-04-2025, Раздел 2**.

## Модули расчёта

### 1. Буровые работы
```typescript
import { calculateDrilling } from '@/modules/calculations/geological';

const result = await calculateDrilling({
  drillingMethod: 'light_mechanical',
  soilCategory: 'III',
  wellsCount: 5,
  depthPerWell: 15,
  season: 'winter',
  waterSaturation: 'saturated'
});
```

### 2. Лабораторные испытания
```typescript
import { calculateLaboratory } from '@/modules/calculations/geological';

const result = await calculateLaboratory({
  tests: [
    { testType: 'moisture', samplesCount: 20 },
    { testType: 'density', samplesCount: 20 },
    { testType: 'compression', samplesCount: 10 }
  ],
  urgency: 'normal'
});
```

### 3. Полевые испытания
```typescript
import { calculateFieldTesting } from '@/modules/calculations/geological';

const result = await calculateFieldTesting({
  testType: 'static_penetration',
  soilCategory: 'II',
  pointsCount: 8,
  depthPerPoint: 12,
  season: 'summer'
});
```

## Категории грунтов

| Категория | Описание | Примеры |
|-----------|----------|---------|
| **I** | Лёгкие грунты | Песок, супесь |
| **II** | Средние грунты | Суглинок мягкопластичный |
| **III** | Тугопластичные | Суглинок тугопластичный, глина мягкопластичная |
| **IV** | Твёрдые связные | Глина тугопластичная |
| **V** | Очень твёрдые | Глина твердая, мерзлые грунты |
| **VI** | Скальные | Известняк, песчаник, скальные породы |

## Методы бурения

### Ручное бурение
- Глубина: до 20 м
- Применение: разведка, мелкие объекты
- Цена: 625-3,750 ₸/м

### Лёгкое механизированное
- Глубина: до 30 м
- Установки: УРБ-2А, ПБУ
- Цена: 1,250-5,000 ₸/м

### Тяжёлое механизированное
- Глубина: до 50+ м
- Установки: ЗИФ-1200, УКБ-12/25
- Цена: 2,500-6,875 ₸/м
